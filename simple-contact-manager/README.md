# 迷你电话簿管理工具 simple-contact-manager

---

## 功能

1. Save to file
2. Load from file
3. Add
4. Search by name
5. Delete
6. Modify
7. List all
8. Clear all
9. Sort by name

### 配置

1. 将项目文件克隆至Linux设备

```bash
git clone https://github.com/glerium/projects-2023.git
```

2. 切换到项目目录

```bash
cd projects-2023/simple-contact-manager
```

3. 编译源代码

```bash
make
```

此时当前目录下将编译生成程序文件contact

4. 运行程序

```bash
./contact
```
